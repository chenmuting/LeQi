# 乐栖 LeQi

乐栖 LeQi 是一个面向 Windows 桌面的音乐播放器，支持在线音乐搜索、发现页、歌单分类、歌手歌曲、专辑页、播放队列、本地歌单、用户歌单、收藏、播放历史统计、主题切换、数据备份与应用内更新下载。

本仓库是 **乐栖 LeQi 的公开发布仓库**，用于提供安装包、绿色版压缩包和更新检查文件。源码仓库为私有仓库，不在此处公开。

---

## 主要功能

### 在线音乐播放

- 支持通过关键词搜索歌曲、歌手、专辑。
- 支持搜索结果分页加载更多。
- 支持搜索页歌曲表使用 `QTableView + QAbstractTableModel` 试点优化。
- 支持双击搜索结果直接播放。
- 支持播放链接获取失败提示。
- 支持播放失败自动停止当前播放后端，避免界面卡顿。
- 支持连续播放失败次数治理，达到上限后停止自动跳过。

### 发现页

- 聚合快速搜索、搜索历史、热门歌单分类、歌手入口和 API 能力检测。
- 支持从发现页一键进入搜索页、歌手页和歌单分类页。
- 支持 API 能力检测，便于判断当前 API 服务是否支持搜索、播放链接、歌单分类和歌手歌曲等接口。

### 搜索历史与批量加入歌单

- 自动保存最近 20 个搜索关键词。
- 支持从搜索页或发现页重新搜索历史关键词。
- 支持将已加载的搜索结果批量加入指定本地歌单。
- 重复歌曲会自动跳过。

### 歌手与歌手歌曲

- 支持搜索歌手。
- 支持查看歌手详情和歌手简介。
- 支持分页加载歌手歌曲。
- 支持在已加载歌手歌曲中本地搜索。
- 支持播放、收藏、加入指定本地歌单。
- 支持从歌手歌曲右键打开专辑。

### 专辑页

- 支持搜索专辑。
- 支持查看专辑详情。
- 支持加载专辑歌曲。
- 支持播放、收藏、加入指定本地歌单。
- 支持将整张专辑歌曲批量加入本地歌单。
- 支持从搜索结果右键打开专辑。
- 支持从歌手歌曲右键打开专辑。

### 歌单分类

- 支持调用 `/playlist/catlist` 获取在线歌单分类。
- 支持按 `/playlist/catlist` 返回的 category 信息分组显示分类标签。
- 支持调用 `/top/playlist` 按分类分页浏览网友精选歌单。
- 支持双击歌单加载歌单详情。
- 支持将当前歌单歌曲批量加入本地歌单。

### 播放控制

- 播放 / 暂停 / 继续。
- 上一首 / 下一首。
- 播放进度拖动。
- 音量调节。
- 播放模式切换。
- 系统托盘控制。

### 播放队列

- 查看当前播放队列。
- 播放选中歌曲。
- 移除队列歌曲。
- 清空播放队列。
- 支持拖拽排序。
- 支持上移、下移、置顶、置底。
- 将当前播放队列保存为本地歌单，并保留当前排序。
- 将队列中的单首歌曲加入指定本地歌单。

### 本地歌单管理

- 新建本地歌单。
- 重命名本地歌单。
- 删除本地歌单。
- 查看歌单内歌曲。
- 搜索当前本地歌单内歌曲。
- 支持本地歌单分类、标签和说明。
- 支持按分类筛选本地歌单。
- 从搜索结果、歌手页、专辑页、歌单分类、收藏、播放历史、播放队列中加入指定本地歌单。
- 加入歌曲时会弹出本地歌单选择器，可明确选择目标歌单。
- 支持新建目标歌单。
- 重复歌曲自动跳过，避免重复加入。
- 支持本地歌单导入、导出和去重。

### 用户歌单

- 支持登录后读取用户歌单。
- 登录入口位于设置页。
- 未登录时显示二维码登录区域。
- 登录后显示用户头像和用户信息。
- 支持用户歌单缓存管理。

### 收藏管理

- 支持收藏歌曲。
- 支持查看收藏列表。
- 支持移除收藏。
- 支持将收藏歌曲加入指定本地歌单。

### 播放历史与统计

- 有效播放达到阈值后才记录历史。
- 支持播放历史列表。
- 支持分页加载播放历史，默认显示最近 50 条，可继续加载更多。
- 支持累计播放次数统计。
- 支持清空播放历史与所有播放次数。
- 播放历史歌曲可加入指定本地歌单。

### UI、性能与稳定性

- 页面切换采用延迟刷新，降低切换瞬间卡顿。
- 首页、歌单管理、收藏、播放历史、数据管理等页面采用后台加载。
- 本地音乐扫描延迟到页面显示后执行。
- 数据导入、导出、清理和日志读取在后台执行。
- 表格批量渲染时暂停 UI 更新和信号，降低大列表刷新卡顿。
- 搜索页表格试点升级为 `QTableView + QAbstractTableModel`。
- 本地歌单搜索支持 300ms 防抖。
- v1.0.16 起修复启动阶段高概率闪退问题。
- 新增功能不在启动阶段主动请求网络接口。

### 启动自检中心

- 检查配置文件、版本文件、数据目录、日志目录、应用图标、QSS 样式、SQLite 数据库、本地歌单 JSON 和 API 地址配置。
- 支持导出启动自检报告到 `logs/startup_check_<timestamp>.json`。
- 用于定位启动失败、目录不可写、配置缺失和数据文件损坏等问题。

### 下载管理与应用更新

- 支持在应用内检查版本更新。
- 支持在关于页下载 Windows 安装包。
- 支持在关于页下载 Windows 绿色版压缩包。
- 支持下载前检测 Release 文件是否存在。
- 支持显示 HTTP 错误原因。
- 支持下载完成后校验文件大小。
- 支持打开已下载文件、打开下载目录、复制文件路径。

### 数据管理

- 支持导出用户数据备份。
- 支持导入用户数据备份。
- 导入前可自动备份当前数据。
- 支持清空缓存、日志、下载目录、收藏、本地歌单、用户歌单缓存、播放历史和播放次数。
- 支持打开备份目录。

### 主题系统

内置主题：

- 深色默认
- 清爽浅色
- 暖沙浅色

主题入口位于侧边栏。界面样式统一使用 QSS，便于后续扩展自定义主题。

### 系统诊断与关于页

- 查看运行环境。
- 查看版本信息。
- 检查更新。
- 执行启动自检。
- 导出启动自检报告。
- 复制诊断信息。
- 打开日志目录、配置目录、数据目录、备份目录。
- 查看 GitHub Release 下载信息。

## 基础使用说明

### 1. 搜索并播放歌曲

1. 打开应用。
2. 进入「搜索」。
3. 输入歌曲名、歌手名或专辑名。
4. 点击「搜索」。
5. 点击「加载更多」可继续分页获取结果。
6. 双击搜索结果中的歌曲开始播放。
7. 右键歌曲可加入歌单、查看歌手全部歌曲或打开专辑。

### 2. 使用发现页

进入「发现」页面：

- 可查看最近搜索历史。
- 可快速搜索关键词。
- 可快速打开热门歌单分类。
- 可输入歌手名进入歌手页。
- 可执行 API 能力检测。

### 3. 查看歌手歌曲

进入「歌手」页面：

- 输入歌手名并搜索。
- 双击歌手加载歌曲。
- 点击「歌手详情」查看简介。
- 点击「加载更多歌曲」分页加载。
- 在「歌曲内搜索」中筛选已加载歌曲。
- 右键歌曲可打开专辑。

### 4. 查看专辑

进入「专辑」页面：

- 输入专辑名或歌手名搜索。
- 双击专辑加载专辑歌曲。
- 可播放专辑歌曲。
- 可将单首歌曲或整张专辑加入本地歌单。

### 5. 浏览歌单分类

进入「歌单分类」页面：

- 点击「刷新分类」获取 `/playlist/catlist` 分类。
- 分类会按 category 分组显示。
- 选择分类后点击「加载歌单」。
- 双击歌单加载歌曲。
- 可将当前歌单歌曲批量加入本地歌单。

### 6. 管理播放队列

进入「播放队列」页面：

- 可播放、移除或清空队列歌曲。
- 可拖拽调整播放顺序。
- 可使用上移、下移、置顶、置底调整顺序。
- 保存为本地歌单时会保留当前顺序。

### 7. 加入本地歌单

在以下页面中可将歌曲加入本地歌单：

- 搜索结果
- 歌手页
- 专辑页
- 歌单分类
- 我的收藏
- 播放历史
- 播放队列
- 歌单管理

操作方式：

1. 选中或右键歌曲。
2. 点击「加入指定本地歌单」。
3. 在弹出的歌单选择器中选择目标歌单。
4. 如需新歌单，选择「＋ 新建本地歌单」。
5. 加入成功后不会自动跳转页面。

### 8. 管理本地歌单

进入「歌单管理」：

- 新建歌单
- 重命名歌单
- 编辑分类 / 标签 / 说明
- 按分类筛选歌单
- 删除歌单
- 搜索歌单内歌曲
- 导入本地歌单 JSON
- 导出本地歌单 JSON
- 歌单去重

### 9. 查看播放历史

进入「播放历史」：

- 查看最近播放记录。
- 默认加载最近 50 条历史。
- 点击「加载更多」继续追加显示。
- 查看累计有效播放次数。
- 将历史歌曲加入本地歌单。
- 清空播放历史与所有播放次数。

### 10. 启动自检

进入「系统诊断」页面：

- 点击「启动自检」。
- 查看配置、版本、目录权限、数据库、本地歌单 JSON 和 API 地址检查结果。
- 点击「导出启动自检报告」生成 JSON 报告。

### 11. 导出 / 导入用户数据

进入「数据管理」：

- 点击「导出用户数据」生成备份文件。
- 点击「导入用户数据」恢复备份。
- 默认开启导入前自动备份当前数据。
- 导入、导出、清理、日志读取均在后台执行。

### 12. 检查更新与下载安装包

进入「关于」页面：

- 点击「检查更新」。
- 点击「下载 ZIP 版」。
- 点击「下载安装包 EXE」。
- 下载完成后可点击「打开已下载文件」。
- 点击「打开下载目录」查看下载文件。
- 点击「复制文件路径」复制本地下载路径。

## 数据目录说明

应用运行后会使用本地数据目录保存配置、缓存和用户数据。常见内容包括：

```text
config/app_config.json       应用配置
data/app.db                  本地数据库
data/local_playlists.json    本地歌单
data/search_history.json     搜索历史
data/downloads/              下载目录
data/backups/                用户数据备份
logs/                        日志目录
logs/startup_check_*.json    启动自检报告
```

可在「关于」或「数据管理」中打开相关目录。

## 常见问题

### 下载链接 404

请确认对应版本的 Release 已经上传以下文件：

```text
LeQiSetup-<version>.exe
LeQi-windows.zip
version.json
```

### 无法检查更新

请确认公开仓库 Release 中存在：

```text
version.json
```

并且可通过以下地址访问：

```text
https://github.com/chenmuting/LeQi/releases/latest/download/version.json
```

### 启动闪退

v1.0.16 起已修复启动阶段高概率闪退问题。若仍启动失败，应用会弹出错误提示，并写入：

```text
logs/error.log
```

可进入「系统诊断 -> 启动自检」检查启动依赖。

如果 `data/local_playlists.json` 损坏，应用会自动隔离为：

```text
data/local_playlists.json.broken_<timestamp>
```

### 页面切换卡顿

v1.0.14 起，首页、歌单、收藏、历史、数据管理等页面已改为后台加载。如果仍有卡顿，请检查本地歌单或播放历史数据量，并优先导出备份后清理历史数据。

### API 接口不可用

进入「发现」页面，点击「检测接口能力」，可检测：

```text
/search
/song/url/v1
/playlist/catlist
/top/playlist
/artist/songs
/api/v2/artist/songs
```

### 播放失败或连续跳过

应用会记录播放事件日志。可进入：

```text
数据管理 -> 播放事件日志
```

查看、筛选、导出或清空日志。
