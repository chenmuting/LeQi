# 乐栖 LeQi

乐栖 LeQi 是一个面向 Windows 桌面的音乐播放器，支持在线音乐搜索、播放队列、本地歌单、用户歌单、收藏、播放历史统计、主题切换、数据备份与应用内更新下载。

本仓库是 **乐栖 LeQi 的公开发布仓库**，用于提供安装包、绿色版压缩包和更新检查文件。源码仓库为私有仓库，不在此处公开。

---

## 主要功能

### 在线音乐播放

- 支持通过关键词搜索歌曲、歌手、专辑。
- 支持双击搜索结果直接播放。
- 支持播放链接获取失败提示。
- 支持播放失败自动停止当前播放后端，避免界面卡顿。
- 支持连续播放失败次数治理，达到上限后停止自动跳过。

### 播放控制

- 播放 / 暂停 / 继续。
- 上一首 / 下一首。
- 播放进度拖动。
- 音量调节。
- 播放模式切换。
- 系统托盘控制。

### 播放队列

- 查看当前播放队列。
- 播放选中歌曲。
- 移除队列歌曲。
- 清空播放队列。
- 将当前播放队列保存为本地歌单。
- 将队列中的单首歌曲加入指定本地歌单。

### 本地歌单管理

- 新建本地歌单。
- 重命名本地歌单。
- 删除本地歌单。
- 查看歌单内歌曲。
- 搜索当前本地歌单内歌曲。
- 从搜索结果、收藏、播放历史、播放队列中加入指定本地歌单。
- 加入歌曲时会弹出本地歌单选择器，可明确选择目标歌单。
- 支持新建目标歌单。
- 重复歌曲自动跳过，避免重复加入。
- 支持本地歌单导入、导出和去重。

### 用户歌单

- 支持登录后读取用户歌单。
- 登录入口位于设置页。
- 未登录时显示二维码登录区域。
- 登录后显示用户头像和用户信息。
- 支持用户歌单缓存管理。

### 收藏管理

- 支持收藏歌曲。
- 支持查看收藏列表。
- 支持移除收藏。
- 支持将收藏歌曲加入指定本地歌单。

### 播放历史与统计

- 有效播放达到阈值后才记录历史。
- 支持播放历史列表。
- 支持累计播放次数统计。
- 支持清空播放历史与所有播放次数。
- 播放历史歌曲可加入指定本地歌单。

### 下载管理与应用更新

- 支持在应用内检查版本更新。
- 支持在关于页下载 Windows 安装包。
- 支持在关于页下载 Windows 绿色版压缩包。
- 支持下载前检测 Release 文件是否存在。
- 支持显示 HTTP 错误原因。
- 支持下载完成后校验文件大小。
- 支持打开已下载文件、打开下载目录、复制文件路径。

### 数据管理

- 支持导出用户数据备份。
- 支持导入用户数据备份。
- 导入前可自动备份当前数据。
- 支持清空缓存、日志、下载目录、收藏、本地歌单、用户歌单缓存、播放历史和播放次数。
- 支持打开备份目录。

### 主题系统

内置主题：

- 深色默认
- 清爽浅色
- 暖沙浅色

主题入口位于侧边栏。界面样式统一使用 QSS，便于后续扩展自定义主题。

### 系统诊断与关于页

- 查看运行环境。
- 查看版本信息。
- 检查更新。
- 复制诊断信息。
- 打开日志目录、配置目录、数据目录、备份目录。
- 查看 GitHub Release 下载信息。

## 安装与启动

### 方式一：安装包

下载：

```text
LeQiSetup-1.0.13.exe
```

双击运行安装包，按安装向导完成安装。适合普通用户。

下载：


解压后运行：

```text

```

适合不想安装到系统目录的用户。

## 基础使用说明

### 1. 搜索并播放歌曲

1. 打开应用。
2. 进入「音乐搜索」。
3. 输入歌曲名、歌手名或专辑名。
4. 点击「搜索」。
5. 双击搜索结果中的歌曲开始播放。

### 2. 加入本地歌单

在以下页面中可将歌曲加入本地歌单：

- 搜索结果
- 我的收藏
- 播放历史
- 播放队列
- 歌单管理

操作方式：

1. 选中或右键歌曲。
2. 点击「加入指定本地歌单」。
3. 在弹出的歌单选择器中选择目标歌单。
4. 如需新歌单，选择「＋ 新建本地歌单」。
5. 加入成功后不会自动跳转页面。

### 3. 管理本地歌单

进入「歌单管理」：

- 新建歌单
- 重命名歌单
- 删除歌单
- 搜索歌单内歌曲
- 导入本地歌单 JSON
- 导出本地歌单 JSON
- 歌单去重

### 4. 查看播放历史

进入「播放历史」：

- 查看最近播放记录。
- 查看累计有效播放次数。
- 将历史歌曲加入本地歌单。
- 清空播放历史与所有播放次数。

### 5. 导出 / 导入用户数据

进入「数据管理」：

- 点击「导出用户数据」生成备份文件。
- 点击「导入用户数据」恢复备份。
- 默认开启导入前自动备份当前数据。

### 6. 检查更新与下载安装包

进入「关于」页面：

- 点击「检查更新」。
- 点击「下载 ZIP 版」。
- 点击「下载安装包 EXE」。
- 下载完成后可点击「打开已下载文件」。
- 点击「打开下载目录」查看下载文件。
- 点击「复制文件路径」复制本地下载路径。

## 数据目录说明

应用运行后会使用本地数据目录保存配置、缓存和用户数据。常见内容包括：

```text
config/app_config.json       应用配置
data/app.db                  本地数据库
data/local_playlists.json    本地歌单
data/downloads/              下载目录
data/backups/                用户数据备份
logs/                        日志目录
```

可在「关于」或「数据管理」中打开相关目录。

## 常见问题

### 下载链接 404

请确认对应版本的 Release 已经上传以下文件：

```text
LeQiSetup-<version>.exe
LeQi-windows.zip
version.json
```

### 无法检查更新

请确认公开仓库 Release 中存在：

```text
version.json
```

并且可通过以下地址访问：

```text
https://github.com/chenmuting/LeQi/releases/latest/download/version.json
```

### 播放失败或连续跳过

应用会记录播放事件日志。可进入：

```text
数据管理 -> 播放事件日志
```

查看、筛选、导出或清空日志。
